# Media & Storage

This section covers object storage and media management solutions.

## Overview

Media and storage services include:

- **Object Storage**: S3-compatible storage
- **File Sharing**: Share files securely
- **Media Servers**: Stream and manage media
- **Backup**: Data backup solutions

## Suggested Services

Platforms to consider:

- **MinIO** - S3-compatible object storage
- **Nextcloud** - File sharing platform
- **Seafile** - File sync and share
- **PhotoPrism** - AI-powered photo management
- **Immich** - Self-hosted Google Photos
- **Jellyfin** - Media streaming server
- **Plex** - Media server
