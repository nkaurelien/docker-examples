# Checkmk

Checkmk is a comprehensive IT infrastructure monitoring solution.

## Features

- Agent-based and agentless monitoring
- 2000+ check plugins
- Auto-discovery
- Alerting and notifications
- Dashboards and reporting
- Distributed monitoring
- REST API

## Quick Start

```bash
cd 05-monitoring-reporting/checkmk
docker compose up -d
```

Access at: `http://localhost:8080/cmk/`

**Credentials:**
- Username: `cmkadmin`
- Password: Your `CMK_PASSWORD`
