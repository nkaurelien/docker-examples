# Uptime Kuma

Uptime Kuma is a self-hosted monitoring tool, an alternative to Pingdom and UptimeRobot.

## Features

- 20+ monitor types (HTTP, TCP, Ping, DNS, Docker, databases)
- 90+ notification services
- Beautiful status pages
- SSL certificate monitoring
- 2FA authentication
- Multi-language support

## Quick Start

```bash
cd 05-monitoring-reporting/uptime-kuma
docker compose up -d
```

Access at: `http://localhost:3001`

## Monitor Types

- HTTP(S), TCP, Ping, DNS
- Docker containers
- MySQL, PostgreSQL, MongoDB, Redis
- Steam, MQTT, gRPC
