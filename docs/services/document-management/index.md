# Document Management

This section covers document management and collaboration platforms.

## Overview

Document management services include:

- **DMS**: Document management systems
- **OCR**: Optical character recognition
- **Collaboration**: Real-time document editing
- **Archival**: Long-term document storage

## Suggested Services

Platforms to consider:

- **Paperless-ngx** - Document management with OCR
- **Nextcloud** - File sharing and collaboration
- **OnlyOffice** - Office document editing
- **Collabora Online** - LibreOffice in the cloud
- **Mayan EDMS** - Enterprise document management
