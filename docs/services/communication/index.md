# Communication

This section covers team communication and collaboration platforms.

## Overview

Communication services include:

- **Chat**: Team messaging
- **Video Conferencing**: Video calls and meetings
- **Forums**: Discussion platforms
- **Notifications**: Alert systems

## Suggested Services

Platforms to consider:

- **Mattermost** - Open source Slack alternative
- **Rocket.Chat** - Team collaboration
- **Zulip** - Chat with threading
- **Matrix/Element** - Decentralized chat
- **Jitsi Meet** - Video conferencing
- **BigBlueButton** - Web conferencing for education
- **Discourse** - Discussion forum
