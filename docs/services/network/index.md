# Network Management

This section covers network infrastructure and communication services.

## Available Services

| Service | Description | Directory |
|---------|-------------|-----------|
| [Asterisk](asterisk.md) | VoIP PBX server | `04-network-management/asterisk/` |

## Overview

Network management services include:

- **VoIP/PBX**: Voice over IP telephony systems
- **DNS**: Domain name resolution
- **VPN**: Virtual private networks
- **Load Balancing**: Traffic distribution
- **Network Monitoring**: Traffic analysis and alerting

## Suggested Services

Additional network services to consider:

- **Pi-hole** - Network-wide ad blocking
- **AdGuard Home** - DNS-based ad blocker
- **WireGuard** - Modern VPN solution
- **OpenVPN** - Full-featured VPN
- **Netdata** - Real-time network monitoring
