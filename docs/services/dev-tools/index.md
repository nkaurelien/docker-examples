# Development Tools

Collection of tools for developers and system administrators.

## Available Tools

### IT-Tools

IT-Tools is a comprehensive collection of handy online tools for developers, created by Corentin Thomasset from Lyon, France.

**Key Features:**
- Network utilities (IP calculator, MAC address tools, IPv6 generator)
- Encoding/Decoding tools (Base64, URL, JWT, etc.)
- Generators (UUID, Hash, Passwords, etc.)
- Converters (JSON, YAML, XML, etc.)
- Text tools (Diff, Regex tester, etc.)
- Development utilities (QR Code, Color picker, etc.)
