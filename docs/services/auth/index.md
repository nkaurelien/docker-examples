# Authentication

Solutions d'authentification et de gestion d'identité.

## Services disponibles

| Service | Description |
|---------|-------------|
| [Keycloak](keycloak.md) | Identity and Access Management |
| [Passbolt](passbolt.md) | Password Manager pour équipes |
| [Zitadel](zitadel.md) | Cloud-native IAM |

## Choisir une solution

- **Keycloak** : Solution mature, très configurable, large écosystème
- **Passbolt** : Gestionnaire de mots de passe open-source pour équipes
- **Zitadel** : Modern, cloud-native, API-first
