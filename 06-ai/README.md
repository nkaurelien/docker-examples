# AI

Artificial Intelligence, Machine Learning, and Data Science platforms.

## Existing Projects

- **jupyter/** - Jupyter notebooks for data science

## Suggested Open Source Services

| Service | Description | GitHub |
|---------|-------------|--------|
| **JupyterHub** | Multi-user Jupyter notebooks | [jupyterhub/jupyterhub](https://github.com/jupyterhub/jupyterhub) |
| **Ollama** | Run LLMs locally | [ollama/ollama](https://github.com/ollama/ollama) |

| **LocalAI** | Self-hosted OpenAI alternative | [mudler/LocalAI](https://github.com/mudler/LocalAI) |

## Quick Start

```bash
cd jupyter/
docker compose up -d
```

Access Jupyter at `http://jupyter.apps.local`.
