# Data Processing

Stream processing, ETL pipelines, and data transformation tools.

## Existing Projects

- **kafka-logstash/** - Kafka cluster with Logstash for real-time data processing

## Quick Start

```bash
cd kafka-logstash/
docker compose up -d
```

Access Kafka UI at `http://localhost:8081`.
