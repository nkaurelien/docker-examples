# IoT and Smart Home

Services for home automation, IoT device management, and website monitoring.

## Existing Projects

- **changedetection/** - Website change detection and monitoring

## Suggested Open Source Services

| Service | Description | GitHub |
|---------|-------------|--------|
| **Home Assistant** | Home automation platform | [home-assistant/core](https://github.com/home-assistant/core) |
| **OpenHAB** | Vendor-neutral home automation | [openhab/openhab-core](https://github.com/openhab/openhab-core) |
| **Domoticz** | Lightweight home automation | [domoticz/domoticz](https://github.com/domoticz/domoticz) |
| **Node-RED** | Flow-based IoT programming | [node-red/node-red](https://github.com/node-red/node-red) |
| **Mosquitto** | MQTT message broker | [eclipse/mosquitto](https://github.com/eclipse/mosquitto) |
| **Zigbee2MQTT** | Zigbee to MQTT bridge | [Koenkk/zigbee2mqtt](https://github.com/Koenkk/zigbee2mqtt) |
| **ESPHome** | ESP8266/ESP32 firmware builder | [esphome/esphome](https://github.com/esphome/esphome) |
| **Scrypted** | Home video integration platform | [koush/scrypted](https://github.com/koush/scrypted) |
| **Frigate** | NVR with AI object detection | [blakeblackshear/frigate](https://github.com/blakeblackshear/frigate) |
| **Gladys Assistant** | Privacy-first smart home | [GladysAssistant/Gladys](https://github.com/GladysAssistant/Gladys) |
| **Uptime Kuma** | Self-hosted monitoring tool | [louislam/uptime-kuma](https://github.com/louislam/uptime-kuma) |

## Quick Start

```bash
cd changedetection/
docker compose up -d
```

Access ChangeDetection at `http://changedetection.apps.local`.
